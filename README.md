## ToDoList UIKit App

This is a simple To-Do List iOS application built using **UIKit**, **MVVM architecture**, and **UserDefaults** for persistent storage. Users can add, delete, and mark tasks as done, with all changes immediately saved and restored across app launches.

## Features
- Add new tasks
- Toggle task completion
- Delete tasks
- Persistent storage using UserDefaults
- Built using UIKit and MVVM pattern
